# 個人記帳系統 (Advanced Personal Expense Tracker)

本專案為大專院校「視窗程式設計 (II)」課程之期末專題。系統採用 C# Windows Forms 進行開發，並透過極致的防呆機制、彈窗互動以及數據視覺化，打造出一款商業級的個人財務管理工具。

## 功能與技術亮點

1. **雙視窗彈窗式互動（Form1 & Form2）**
   - 將新增與編輯功能獨立為動態彈窗，支援修改資料時將舊數據「自動倒填」回 Form2，大幅提升主畫面空間感。
2. **高質感資料網格（DataGridView）**
   - 捨棄傳統 ListBox，全面升級為支援「整列高亮選取」與「禁止多選」的唯讀資料行表格，排版清晰、操作穩定。
3. **動態複合式篩選（即時過濾）**
   - 內建「備註關鍵字（自動轉小寫比對）」與「收支類別下拉選單」雙重過濾機制。打字時畫面即時連動，且總金額會自動重算目前「看得到」的帳目。
4. **智慧預算控管與極致防呆**
   - **金額框鎖定**：利用鍵盤事件攔截，金額欄位與預算欄位「只能輸入數字與退格鍵」，從根本杜絕格式錯誤。
   - **大數支援**：全面升級資料型態至 `long` (64位元)，支援百億級別的天文數字記帳。
   - **智慧預算提示**：當總支出超出設定的預算上限時，會在真正變動資料或按下 Enter 時彈出強烈超支警告，輸入過程中則採字體變紅提示，兼顧操作流暢度。
5. **分頁式視覺化圖表分析（TabControl & Chart）**
   - 利用分頁控制項隔開明細與圖表。統計頁面採用內建 `Chart` 控制項，動態加總各項支出並繪製成精美的「收支比例圓餅圖」。
6. **手動檔案管理與髒資料安全攔截 (Dirty Check)**
   - 內建 `OpenFileDialog` 與 `SaveFileDialog`，使用者可自訂儲存路徑為 `.txt` 或 `.csv` 格式。
   - 若畫面有變動而使用者直接關閉視窗，系統會強制攔截關閉事件，並彈窗詢問是否需要先進行存檔，確保資料不遺失。
